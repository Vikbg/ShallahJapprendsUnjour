# small-engine

Heyyyy it's Viktor !

A small starter OpenGL engine written in C++. It uses GLAD for OpenGL initialization, SDL3 for the windowing system, and [tinyobjloader](https://www.github.com/tinyobjloader/tinyobjloader).

It currently supports model loading with MTL textures and light sources. It also includes some basic keyboard shortcuts, such as `Escape` to quit and `F11` for fullscreen. You can move in world space using WASD/ZQSD controls.

The project is not finished yet. The [engine](include/engine.hpp) needs to be refactored, and additional features still need to be implemented.
